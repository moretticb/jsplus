AJAX.clearParameters( )

Description

Clears the parameters currently stored. Since there is no getter for input parameters, one can check the current state of the data to be sent through the HTTP request using getQueryString method.

Compatibility

Prototype

clearParameters():Void

Returns

None

Parameters

No parameters

Examples

function handleResponse() {}
var ajax = new AJAX("POST", "requestPage.php", true, null, handleResponse);
ajax.setInputParameters({name: "Kyle Smith", age: 22});
alert(ajax.getQueryString()); //Displays name=Kyle%20Smith&age=22
ajax.clearParameters();
alert(ajax.getQueryString()); //Displays null
