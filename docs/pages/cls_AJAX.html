AJAX

Constructor

The list below contains the constructor of the AJAX class:

Constructor Description Compatibility
AJAX() An Asynchronous Javascript and XML (AJAX) feature. It performs HTTP requests through Javascript XMLHttpRequest object.

Methods

The list below contains the methods of the AJAX class:

Method Description Compatibility
setRequestType()Sets the request type of an AJAX instance.
getRequestType()Gets the current request type of an AJAX instance.
setRequestPage()Sets the request page of an AJAX instance.
getRequestPage()Gets the current request page of an AJAX instance.
setAsync()Sets whether or not the request page of an AJAX instance is asynchronous.
isAsync()Gets a boolean that indicates whether or not the request is asynchronous.
setInputParameters()Sets the query string parameters of an AJAX instance. In order to append parameters, avoid the replacement, one can use addParameters method. Since there is no getter for input parameters, one can check the current state of the data to be sent through the HTTP request using getQueryString method.
addParameters()Appends query string parameters of an AJAX instance. In order to avoid unnecessary accumulation of parameters, one can use setInputParameters method. Since there is no getter for input parameters, one can check the current state of the data to be sent through the HTTP request using getQueryString method.
clearParameters()Clears the parameters currently stored. Since there is no getter for input parameters, one can check the current state of the data to be sent through the HTTP request using getQueryString method.
getQueryString()Gets a query string of an AJAX instance. This query string contains every data to be sent through the HTTP request.
setResponseManagerFunction()Sets the function to be called when responses from HTTP requests are received.
getResponseManagerFunction()Gets the function currently assigned to an AJAX instance for handling responses from HTTP requests.
validate()A validator to check whether or not the attributes of an AJAX instance are correctly adjusted to perform HTTP requests.
makeRequest()Performs an HTTP request based on the attributes of an AJAX instance. This method internally calls validate method before issuing an HTTP request.
getResponseText()Gets the response arasing from an HTTP request. Commonly used in synchronous HTTP requests.
getResponseXML()Gets the response arasing from an HTTP request in XML Object format. Commonly used in synchronous HTTP requests. This structure can be explored by accessing childNodes or using methods such as getElementsByTagName().
