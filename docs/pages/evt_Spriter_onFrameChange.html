Spriter.onFrameChange( )

Description

Event triggered when current frame changes. Methods gotoAndPlay and gotoAndStop also trigger this event.

Compatibility

Prototype

onFrameChange(frame:Number):Void

Returns

None

Parameters

Parameter Type Description
frameNumberThe new current frame number.

Examples

Example 1 

var t = new Tween("logo", "width", Quad.prototype.easeInOut, 100, 200, 10, false);
t.onFrameChange = function(frame){
	$('frameIndicator').innerHTML = "Frame #"+frame;
}
