AJAX.getQueryString( )

Description

Gets a query string of an AJAX instance. This query string contains every data to be sent through the HTTP request.

Compatibility

Prototype

getQueryString():String

Returns

A query string

Parameters

No parameters

Examples

function handleResponse() {}
var ajax = new AJAX("POST", "requestPage.php", true, {name: "Kyle Smith", age: 22}, handleResponse);
alert(ajax.getQueryString()); //Displays name=Kyle%20Smith&age=22
