startDrag( )

Description

Function that enables dragging DOM elements. Always called at mousedown event. Use stopDrag function to stop dragging.

Compatibility

Prototype

startDrag(mouseDownEventObject:Object, objId:String, keepZindex:Boolean, axisConstraint:Number, relativity:Number):Boolean

Returns

Returns always false, avoiding the text selection.

Parameters

Parameter Type Description
mouseDownEventObjectObjectThe event variable obtained with mousedown event.
objIdStringThe DOM element ID.
keepZindexBooleanIf true, the z-index CSS property is kept. Otherwise, the next highest value is assigned to this element. Default is true.
axisConstraintNumberAxis restriction constraint. Constants Drag.X_AXIS, Drag.Y_AXIS or Drag.BOTH_AXIS should be used. Default is Drag.BOTH_AXIS.
relativityNumberIndicates whether the positioning is relative or absolute. Constants Drag.RELATIVE or Drag.ABSOLUTE should be used. Default is Drag.ABSOLUTE.

Examples

Example 1 | Example 2 

$("objId").onmousedown=startDrag;
$("objId").onmouseup=stopDrag;
