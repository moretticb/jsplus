SWFObject( )

Description

A faster way to obtain the HTML code of embedded SWF files.

Compatibility

Prototype

SWFObject(swfDir:String, width:Number, height:Number):String

Returns

An HTML string of the specified SWF file to be embedded.

Parameters

Parameter Type Description
swfDirStringUrl of the SWF file.
widthNumberWidth of the embedded area.
heightNumberHeight of the embedded area.

Examples

$('swfContainer').innerHTML = SWFObject('game.swf', 300, 300);
