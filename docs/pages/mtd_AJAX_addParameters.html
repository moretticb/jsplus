AJAX.addParameters( )

Description

Appends query string parameters of an AJAX instance. In order to avoid unnecessary accumulation of parameters, one can use setInputParameters method. Since there is no getter for input parameters, one can check the current state of the data to be sent through the HTTP request using getQueryString method.

Compatibility

Prototype

addParameters(parametersObject:Object):Void

Returns

None

Parameters

Parameter Type Description
parametersObjectObjectThe parameters to be sent to request page. Object properties, as well as their values are internally converted and respectively assigned to parameters and values of a query string.

Examples

function handleResponse() {}
var ajax = new AJAX("POST", "requestPage.php", true, null, handleResponse);
ajax.setInputParameters({name: "Kyle Smith"});
alert(ajax.getQueryString()); //Displays name=Kyle%20Smith
ajax.addParameters({age: 22});
alert(ajax.getQueryString()); //Displays name=Kyle%20Smith&age=22
