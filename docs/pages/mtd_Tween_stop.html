Tween.stop( )

Description

Stops the tweening.

Compatibility

Prototype

stop(callOMFF:Boolean):Void

Returns

None

Parameters

Parameter Type Description
callOMFFBooleanDetermines whether or not the onMotionFinished event is triggered after stopping the tweening.

Examples

Example 1 

window.onload = function(){
	var t = new Tween("logo", "width", Quad.prototype.easeInOut, 100, 200, 100, false);
	
	$('btn').onclick = function(){
		t.stop();
	}
}
