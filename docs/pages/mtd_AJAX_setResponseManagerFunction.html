AJAX.setResponseManagerFunction( )

Description

Sets the function to be called when responses from HTTP requests are received.

Compatibility

Prototype

setResponseManagerFunction(rmFunction:Function):Void

Returns

None

Parameters

Parameter Type Description
rmFunctionFunctionThe function to be called for handling responses from HTTP requests.

Examples

function handleResponse() {}
function anotherFunction() {}
var ajax = new AJAX("POST", "requestPage.php", true, null, handleResponse);
alert(ajax.getResponseManagerFunction()==anotherFunction); //Displays false
ajax.setResponseManagerFunction(anotherFunction);
alert(ajax.getResponseManagerFunction()==anotherFunction); //Displays true
