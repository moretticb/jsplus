Tween.Tween( )

Description

An interpolation feature used to animate visual elements from a CSS property. For simultaneous tweenings of more than one CSS property, create several Tween instances, respectively related to each CSS property.

Compatibility

Prototype

Tween(DOMObjId:String, style:String, easeType:Function, initValue:Number, finalValue:Number, duration:Number, useSeconds:Boolean, xtraProp:Object)

Parameters

Parameter Type Description
DOMObjIdStringThe DOM Element id
styleStringThe CSS property to be tweened
easeTypeFunctionThe tween behavior function. None, Back, Elastic, Quad, Cubic, Quartic, Quintic, Sine, Expo and Circ are examples of the available behaviors.
initValueNumberInitial value
finalValueNumberFinal value
durationNumberThe duration of the tween in frames
useSecondsBooleanIf true, duration is considered as seconds. Otherwise, duration is considered as frames
xtraPropObjectCSS properties to be set before the tweening starts

Examples

Example 1 

new Tween("userPicture", "width", Quad.prototype.easeInOut, 100, 200, 10, false, {visibility: 'visible'}); //Object's width tweening that lasts 10 frames
new Tween("userPicture", "opacity", Quad.prototype.easeInOut, 0, 1, f2m(getFPS(), 10)/1000, true); //Object's opacity tweening that lasts the equivalent of 10 frames in seconds
