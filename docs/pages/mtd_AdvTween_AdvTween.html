AdvTween.AdvTween( )

Description

A Tween based feature used to interpolate a value that can be used to animate one or more elements.

Compatibility

Prototype

AdvTween(easeType:Function, initValue:Number, finalValue:Number, duration:Number, useSeconds:Boolean, theFunction:Function)

Parameters

Parameter Type Description
easeTypeFunctionThe tween behavior function. None, Back, Elastic, Quad, Cubic, Quartic, Quintic, Sine, Expo and Circ are examples of the available behaviors.
initValueNumberInitial value
finalValueNumberFinal value
durationNumberThe duration of the tween in frames
useSecondsBooleanIf true, duration is considered as seconds. Otherwise, duration is considered as frames
theFunctionFunctionThe function to be executed every since the value changes. The current state of the interpolated value can be retrieved by the first parameter of this function.

Examples

Example 1 | Example 2 

new AdvTween(Quad.prototype.easeInOut, 100, 200, 10, false, function(value){
	$('objA').width = value+"px";
	$('objB').width = value+"px";
});
