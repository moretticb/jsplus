Spriter

Constructor

The list below contains the constructor of the Spriter class:

Constructor Description Compatibility
Spriter() A sprite animator feature. Given the sprite image, total number of tiles, number of tiles per line and tile dimensions, this feature produces an animation using FPS variable as the FPS rate.

Methods

The list below contains the methods of the Spriter class:

Method Description Compatibility
stop()Stops playhead displaying the current frame.
play()Activates the playhead displaying the animation from the current frame. If the animation is already playing, nothing happens.
isStopped()Indicates whether or not the animation is stopped.
gotoAndStop()Goes to a frame and stops the animation.
gotoAndPlay()Goes to a frame and activates the playhead.

Events

The list below contains the events of the Spriter class:

Event Description Compatibility
onFrameChange()Event triggered when current frame changes. Methods gotoAndPlay and gotoAndStop also trigger this event.
